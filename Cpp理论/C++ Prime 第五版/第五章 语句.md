# **try语句块和异常处理**

- `throw` 表达式：抛出一个异常

- `try` 语句块：语句块中抛出的异常被 `catch` 子句捕获处理

- 匹配不到 `catch` 模块时，程序转到 `terminate` 标准库函数，执行非正常退出
