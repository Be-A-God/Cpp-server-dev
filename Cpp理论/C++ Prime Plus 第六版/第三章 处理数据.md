# **内置类型**

    基本类型：整数、浮点数

    复合类型：数组、字符串、指针和结构

---

# **整型**

    short至少16位

    int至少和short一样长

    long至少32位，且至少和int一样长

    long long至少64位，且至少和long一样长

---

# **变量初始化**

### 函数内部定义的变量未初始化，则初值不确定

### C++ 其他初始化方式

``` C++
int wrens(33);
int hamburgers = {33};
int emus{2};
// 大括号为空是初始化为零
```
